# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Floatnote.

## Global

These shortcuts work from any application:

| Key | Action |
|-----|--------|
| Cmd+Shift+D | Toggle Floatnote visibility |

## Tools

| Key | Action |
|-----|--------|
| D | Draw mode (pen) |
| T | Add text overlay |
| E | Eraser mode |
| S | Shape tool |
| F | Toggle freeze view |

## Canvas

| Key | Action |
|-----|--------|
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Cmd+Plus | Zoom in |
| Cmd+Minus | Zoom out |
| Cmd+0 | Reset zoom and position |
| Escape | Deselect / exit current mode |
| Delete / Backspace | Delete selected shape |

## Notes

| Key | Action |
|-----|--------|
| Cmd+N | New note |
| Cmd+W | Delete current note |
| Left Arrow | Previous note |
| Right Arrow | Next note |

## File

| Key | Action |
|-----|--------|
| Cmd+E | Export as PNG |
| Cmd+Shift+C | Clear canvas |
| Cmd+V | Paste from clipboard |

## Trackpad Gestures

| Gesture | Action |
|---------|--------|
| Pinch | Zoom in/out |
| Two-finger drag | Pan canvas |
| Two-finger rotate | Rotate canvas |
